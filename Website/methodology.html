Methodology

Source, Cleaning And Assumption Notes

This page holds the detailed notes behind the dashboard so the main experience stays focused on analytics and storytelling. It clarifies what is raw-style metadata, what is engineered, what is modeled, and where the current dataset still needs cleanup.

Source Layer
Master dataset 256 match rows spanning 2010, 2014, 2018 and 2022 in a custom JSON master file.
Observed-style fields Year, date, teams, stadium, city, attendance, capacity and stage are treated as the closest thing to raw event descriptors.
Modeled fields Viewers, suspense, benchmark fields and all 2026 projections should be read as analytical estimates, not official rights-holder measurement.
Cleaning Decisions
Standardized stages Stage labels are normalized so cross-year comparisons stay consistent across overview, stage and forecast modules.
Derived summaries Year totals, stage averages, segmentation and team audience summaries are recalculated directly from the dataset inside the dashboard.
Forecast assumptions 2026 views blend historical audience behavior with team quality, popularity, recent form and expected star availability.
Field Dictionary
Raw-style year, date, home, away, stadium, city, attendance, capacity, stage, host
Engineered att_ratio, is_host, is_opening, is_final, extra_time, penalties
Modeled viewers, suspense, avg_benchmark, final_benchmark and all 2026 scenario outputs

Assumptions
2026 format assumption Forecasts assume a 48-team, 104-match tournament with a wider premium inventory ladder.
Signal assumption Match-level 2026 projections use team quality, brand pull, recent form and expected star availability as audience multipliers.
Honesty boundary The project does not claim streaming-share growth, regional growth, or COVID audience causality because those drivers are not present in the underlying file.
Current Caveats
Attendance quality is not fully clean Several rows show attendance above stated capacity, which means fill-rate logic should be treated as directional rather than definitive until corrected.
Some audience values appear synthetic Repeated rounded viewer values in blocks suggest formula-generated estimates in parts of the dataset, so the project is best framed as hybrid historical-plus-modeled analytics.
Boolean consistency still needs repair Final-stage rows and related flags should be cleaned in a future dataset revision so filter logic stays fully trustworthy.
Qatar 2022 Context
Late-pandemic operating environment Qatar 2022 took place in a still-post-pandemic global environment, with travel and public-health rules being adjusted close to kickoff.
What this project can say The tournament should be interpreted with public-health and travel-policy context in mind, especially when comparing it with earlier editions.
What this project does not claim This dashboard does not attribute 2022 audience changes directly to COVID because the underlying file does not contain market-level, platform-level or causal evidence for that conclusion.
